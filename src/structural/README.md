Structural Patterns
===================

Structural patterns are design patterns which ease the design by identifying a simple way to realize relationships between entities.

List of creational patterns
---------------------------

* `Adapter or Translator`
* `Bridge`
* `Composite`
* `Decorator`
* `Facade`
* `Flyweight`
* `Front Controller`
* `Module`
* `Proxy`
* `Twin`
* `Extensibility`
* `Pipes and Filters`
* `Private class data`
